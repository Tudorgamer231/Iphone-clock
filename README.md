# 🕐 iPhone Clock
 
O replică a modului **StandBy / Clock** de pe iPhone, construită ca un singur fișier HTML — fără dependențe, fără build tools.
 
![HTML](https://img.shields.io/badge/HTML-single--file-orange) ![CSS](https://img.shields.io/badge/CSS-vanilla-blue) ![JS](https://img.shields.io/badge/JS-vanilla-yellow)
 
## ✨ Features
 
- **Ceas digital** cu animații flip fluide la schimbarea cifrelor
- **Ceas analogic** cu ace animate smooth
- **Setări live** — culoare, grosime font, mărime font, toggle digital/analogic
- **Status bar** cu baterie reală (unde e suportat de browser)
- **System Info panel** — rezoluție, pixel ratio, conexiune, OS, browser
- Animație de „breathing" subtilă
- Colon care clipește în ritmul secundelor
- Panou de setări care apare la click și dispare automat după 5 secunde
## 🚀 Cum îl folosești
 
Nu necesită instalare. Deschide direct în browser:
 
```bash
# Clonează repo-ul
git clone https://github.com/tudustech/iphone-clock.git
 
# Deschide fișierul
open index.html
```
 
Sau accesează live pe **GitHub Pages**: `https://tudustech.github.io/iphone-clock`
 
## 🎨 Personalizare
 
Click oriunde pe ecran pentru a deschide panoul de setări:
 
| Opțiune | Detalii |
|---|---|
| Culoare | 8 culori predefinite (alb, galben, portocaliu, albastru, verde, roz, roșu, violet) |
| Grosime font | Slider de la Thin (100) la Heavy (900) |
| Mărime font | Slider de la 8vw la 28vw |
| Mod analogic | Toggle între digital și analogic |
